$$a(\mathcal{N}) = a_0*e^{\frac{\mathcal{N}^2}{2}}$$

$$a(\eta) = a_0*(1+(\frac{\eta}{\eta_0})^2)^p$$

$$\eta^+(\mathcal{N}) = \eta_0\left[\left(\frac{a(\mathcal{N})}{a_0}\right)^{\frac{1}{p}}-1\right]^{\frac{1}{2}}$$ $$\eta^-(\mathcal{N}) = -\eta_0\left[\left(\frac{a(\mathcal{N})}{a_0}\right)^{\frac{1}{p}}-1\right]^{\frac{1}{2}}$$

$$\eta(\mathcal{N}) = \theta(\mathcal{N})*\eta^+(\mathcal{N}) + \theta(\mathcal{-N})*\eta^-(\mathcal{N})$$

$$a(\mathcal{N}) = a(\eta(\mathcal{N}))$$

$$\mathcal{H}(\eta) = \frac{a^{'}(\eta)}{a(\eta)^2}$$

function prefix of 'd' for differentiation with respect to $\eta$ and 'D' for differentiation with respect to $\mathcal{N}$. Lower case function names for functions in terms of $\eta$ and higher case function names for functions in terms of $\mathcal{N}$
